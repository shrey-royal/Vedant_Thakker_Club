Control Statements:-
	-> these statements controls the flow of the program.

1. if statement:-
	-> used when the program have to make a decision.
4 ways of using these if statements
	1. If statement Only
	-> syntax:-
		if(condition){
			//body of the if statement
		}

	2. if else statement
	->syntax:-
		if(condition){
			//if block
		}else{
			//body of else block
		}

	3. if else if statement (ladder if statement)
	->syntax:-
		if(condition_1){
			//if block
		}else if(condition_2){
			//else block
		}else if(condition_3){
			//else block
		}else{
			//else block
		}


	4. nested if statements
	-> syntax:-
		if(condition){
			if(condition){	//nested if
				//nested if block
			}else{
				//nested else block
			}
		}else{
			//else block
		}
